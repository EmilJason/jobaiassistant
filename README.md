# AI Job Application Assistant

A starter full-stack web app for reviewing resumes, generating cover letters, tracking job applications, and preparing for interviews.

## Stack

- React + Vite for the frontend
- Express for the backend API
- OpenAI-ready server structure for future AI features

## Quick start

1. Install dependencies:
   npm install
2. Start the app:
   npm run dev
3. Open the frontend at http://localhost:5173
4. API health check: http://localhost:3001/api/health
